About RepoGuardian

Our Mission

RepoGuardian is designed to help developers and organizations identify and eliminate security vulnerabilities in their code repositories before they become costly breaches.

In today's fast-paced development environment, it's easy for sensitive information like API keys, database credentials, and private keys to accidentally end up in version control. Our advanced scanning engine helps you catch these issues early.

How It Works

1. Submit Your Code

Upload a ZIP file or provide a Git repository URL for scanning.

2. Advanced Scanning

Our regex-based engine analyzes your code for various types of secrets and sensitive data.

3. Detailed Reports

Receive comprehensive reports with file locations, line numbers, and severity levels.

4. Take Action

Use our findings to secure your codebase and prevent security breaches.

What We Detect

  • AWS Access Keys & Secrets
  • Google Cloud Platform Keys
  • JWT Tokens
  • Database Connection URLs
  • Email/Password Combinations
  • Private Keys (.pem, .key)
  • API Keys
  • OAuth Tokens
  • SSH Keys
  • And much more...

Security & Privacy

We take your privacy seriously. All uploaded files and cloned repositories are:

  • Processed in secure, isolated environments
  • Automatically deleted after scanning
  • Never stored permanently on our servers
  • Transmitted using encrypted connections

Only scan results and metadata are retained to provide you with historical tracking and reporting capabilities.

Technology Stack

RepoGuardian is built using modern, reliable technologies:

  • Backend: Django (Python)
  • Database: SQLite/PostgreSQL
  • Git Integration: GitPython
  • Frontend: Bootstrap 5
  • Reports: WeasyPrint (PDF)
  • Email: Django Email Framework
